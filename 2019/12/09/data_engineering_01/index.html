Pipes and Filters

cat : 해당 파일 전체를 print
head : 해당 파일 앞의 10줄 정도를 print
tail : 해당 파일 뒤의 20줄 정도를 print

command > file : 기존의 파일 내용은 지우고 현재 command한 결과 파일에 저장
command >> file : 기존의 파일에 덮붙여서 결과를 저장(python append같은 느낌!)
